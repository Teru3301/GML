Искусственный интеллект в данном проекте представляется в виде последовательности исполняемых «генов». 

Каждый ген — это настраиваемая структура, содержащая исполняемую функцию, механизм мутации и набор параметров.

В теории этот подход должен стать более гибкой заменой классическим нейронным сетям.
